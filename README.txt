Cambiar el EMAIL_HOST en setting.py por el email de la empresa,

En la sección de soporte poner el email con el que se van a realizar los contactos para soporte tecnico asi como el telefono, 

En el admin en la sección de Sitios cambiar el nombre de dominio que aparece de ejemplo por el del dominio donde se aloje la página.

En caso de desarrollo:
Clonar el repo
Ejecutar pip install -r requirement.txt
